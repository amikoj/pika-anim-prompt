## The Ultimate AI Anime Art Recipe Book

This repository houses the complete recipe library for Pikanim — the browser-based AI tool that transforms your photos, game screenshots, and cosplay shots into stunning anime art in seconds.

### What's Inside

Every `.md` file in the `posts/` directory is a step-by-step guide that teaches you how to get pro-level anime conversions from everyday photos. These posts power the [Pikanim Blog](https://www.pikanim.com/blog) — a growing collection of practical, no-jargon guides for turning real-life moments into anime masterpieces.

### Why Open Source?

Because great art recipes should be shared. If you've discovered an amazing photo-to-anime workflow, a perfect prompt pairing, or a killer before/after comparison, open a PR and add it to the cookbook. Let's build the world's best open collection of AI anime style recipes together.

### How to Use These Recipes

Each recipe article links directly to the corresponding Pikanim style page. Just click through, drop your photo, and see the transformation happen in your browser. No downloads. No GPU. No coding.
